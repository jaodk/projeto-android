# Projeto Android

Este é um site simples desenvolvido como parte de um exercício de HTML e CSS. O objetivo do projeto é apresentar informações sobre o sistema operacional Android, utilizando boas práticas de estruturação semântica e design responsivo.

## 🔍 Descrição

O **Projeto Android** simula uma página institucional com:

- Estrutura HTML semântica (`header`, `main`, `section`, `footer`, etc.)
- Estilo personalizado com CSS
- Fonte personalizada nos títulos (`idroid.otf`)
- Design responsivo para diferentes tamanhos de tela
- Imagens e vídeos relacionados ao Android

## 💻 Tecnologias Utilizadas

- HTML5
- CSS3
- Fonte personalizada (`idroid.otf`)
- Google Fonts (`Poppins`, `Orbitron`, etc.)
- Layout responsivo com media queries
